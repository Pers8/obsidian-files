# 6. Plan de Test 

| Success Criteria                                                                 | Description of the Test                                                                                         | Test Method                                                                                                                    | Expected Outcome                                                                                                                              |
| -------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------- |
| 1. L'utilisateur peut voir les prix et tous mes travaux sur un seul site web.    | Verify that the website displays all available services with their prices and showcases the portfolio of works. | Navigate through the website to the sections displaying prices and past works.                                                 | All services offered are listed with clear pricing. The portfolio section displays a variety of past works.                                   |
| 2. Le site est automatisé et lié à un serveur Discord.                           | Test the integration between the website and the Discord server for automated notifications or actions.         | Perform an action on the website that should trigger a notification or action on the Discord server (e.g., submitting a form). | The Discord server receives the notification or performs the action as expected, indicating successful integration.                           |
| 3. Les commandes peuvent être gérées et créées directement à partir du site web. | Ensure that users can place orders through the website and that these orders are correctly processed.           | Place an order on the website, filling out all necessary information and submitting it.                                        | The order is successfully created and reflected in the backend system or database, and a confirmation is displayed or sent to the user.       |
| 4. L'utilisateur peut facilement commander plusieurs articles à la fois.         | Test the website's functionality to handle multiple item orders in a single transaction.                        | Add multiple items to a cart or order form on the website and submit the order.                                                | The website processes the order correctly, including all items. A summary or confirmation showing all items ordered is presented to the user. |

| -------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

L'utilisateur peut voir les prix et tous mes travaux sur un seul site web.    | Vérifier que le site web affiche tous les services disponibles avec leurs prix et présente le portfolio des travaux.         | Naviguer sur le site web jusqu'aux sections affichant les prix et les travaux antérieurs.                                                                 | Tous les services proposés sont listés avec une tarification claire. La section portfolio affiche une variété de travaux antérieurs.                                      |
| 2. Le site est automatisé et lié à un serveur Discord.                           | Tester l'intégration entre le site web et le serveur Discord pour les notifications ou actions automatisées.                 | Effectuer une action sur le site web qui devrait déclencher une notification ou une action sur le serveur Discord (par exemple, soumettre un formulaire). | Le serveur Discord reçoit la notification ou effectue l'action comme prévu, indiquant une intégration réussie.                                                            |
| 3. Les commandes peuvent être gérées et créées directement à partir du site web. | S'assurer que les utilisateurs peuvent passer des commandes via le site web et que ces commandes sont correctement traitées. | Passer une commande sur le site web, en remplissant toutes les informations nécessaires et en la soumettant.                                              | La commande est créée avec succès et reflétée dans le système de gestion ou la base de données, et une confirmation est affichée ou envoyée à l'utilisateur.              |
| 4. Un résumé ou une confirmation montrant tous les articles commandés est présenté à l'utilisateur. |
